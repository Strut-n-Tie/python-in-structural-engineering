Welcome to the **Python in Structural Engineering** course! This repository contains all the code examples and Jupyter notebooks from the video series on the [Strut'n'Tie](https://www.youtube.com/@Strut_n_Tie "Strut'n'Tie") YouTube channel.

## About This Course

No coding experience required! This course introduces structural engineers to Python in a friendly, practical way. You'll learn to move beyond spreadsheets and boost your productivity using Python, all through examples you'd actually encounter on real engineering jobs.

##  Repository Structure

For each session, you'll find two folders (where applicable):

- **`start/`** - An initial notebook to fill in the code yourself
- **`end/`** - The complete version to explore and experiment with

## Course Content

- Why structural engineers should learn Python
- Setting up your environment (VS Code, Jupyter Notebooks)
- Markdown for formatting calculations
- Python basics (variables, lists, dictionaries)
- Essential libraries:
  - **Matplotlib** – for generating graphs
  - **NumPy** – for matrix calculations
  - **SymPy** – for symbolic mathematics
  - **Pandas** – for table-like data

## Getting Started

You can get the course materials in two ways:

### Option 1: Clone with Git (recommended for updates)

```
git clone https://github.com/Strut-n-Tie/python-in-structural-engineering.git
```

This creates a copy on your computer and lets you easily pull future updates.

### Option 2: Download ZIP (simplest)

1. Go to [github.com/Strut-n-Tie/python-in-structural-engineering](https://github.com/Strut-n-Tie/python-in-structural-engineering)
2. Click the green **"Code"** button
3. Select **"Download ZIP"**
4. Extract the ZIP file to your preferred folder

### After you have the files:

- Navigate to the session you're interested in
- Open the `.ipynb` files in Jupyter Notebook or VS Code
- Follow along with the videos!

>  **Note:** Check modification dates as this repository is continually updated. If you cloned the repository, you can get updates with `git pull`.
